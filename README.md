# Baikal_poptranscriptomics

Helper scripts for the transcriptome-base population genetics project.

Samples: *E. verrucosus* from Bolshie Koty (BK), Port Baikal (PB) and Ust-Barguzin (UB) and *E. cyaneus* from BK and PB.
